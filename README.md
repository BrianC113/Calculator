PROJECT TITLE: 
Simple Calculator

PURPOSE OF PROJECT: 
This project is a simple calculator application built using JavaFX. 
It will demonstrate the Model-View-Controller(MVC) method where Model performs the calculations, View handles the user interface/what you'll see and the Controller handles the user's input. 
This calculator will perform simple mathematics such as addition, subtraction, multiplication and division.

VERSION or DATE: 
January 2026

HOW TO START THIS PROJECT:
1. Open the project in a Java IDE that supports JavaFX (e.g. BlueJ, IntelliJ, or Eclipse).
2. Ensure JavaFX is properly configured in your environment.
3. Run the main application class that launches the JavaFX window.
4. The calculator window will open automatically.

USER INSTRUCTIONS:
1. Enter a whole number in the first input field.
2. Enter a whole number in the second input field.
3. Click one of the operator buttons i.e - Add(+), Subtract(-), Multiply(x), Divide(/).
4. The result of the calculation will be displayed in the result field.
5. If invalid input is entered, an error message will be shown instead.
